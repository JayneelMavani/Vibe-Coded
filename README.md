# Vibe-Coded

A small collection of web projects and demos.

Included folders:

- Aloysius-Technologies
- BookHub
- GreenRoots
- Phising-Link-Detector (backend + frontend)
- Tada

How to use:

- Open any project's `index.html` in a browser for the frontend demos.
- For `Phising-Link-Detector`, see `Phising-Link-Detector/backend` to run the Node.js server.

That's all — simple demos and starter code for learning and quick previews.
